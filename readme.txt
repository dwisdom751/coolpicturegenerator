To use this program, run the changeSizeCan.sh bash script 
and input a height and width as prompted to 
generate a canvas of the size you wish to generate.
This will make a new processing file in its own directory that will be 
ready to run.

The progam will generate a random shapes on a canvas.
The canvas can be saved as a png image file by pressing s at any time
while the program is running. The program can switched to generate rectangles
instead by pressing r at any time. The program can be switched back to 
generate ellipses instead again by pressing e. The canvas can be cleared by
pressing n at time while the program is running. The shapes can be generated
can be made larger by pressing m at any time while the program is running
and made smaller by pressing l at any time while the program is running.
